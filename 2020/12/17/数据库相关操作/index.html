数据库相关操作

Thu Dec 17 2020

DDL - 数据定义语言

创建数据库

create database 数据库名 default charset utf8 (default 后面的语句，是修改字符集)

删除数据库

drop 数据库名

切换数据库

use 数据库名

创建表

create table 表名
(
字段名1 类型 约束,

字段名2 类型 约束,

…

字段名n 类型 约束,
primary key(主键名)

);

注：primary key 设置主键

删除表

drop table 表名；

修改表

添加字段

alter table 表名 add column 字段名 类型 约束；

添加外键

alter table 表名add constraint 外键名
foreign key (外键) references 关联的外键表 (关联的外键名);

DML - 数据操作语言

添加数据

  • 按位依次添加

  • 按字段名添加

  • 批量添加

更新数据

update 表名 set 字段名1=修改的内容1，字段名2=修改的内容2 where 条件；

删除数据

delete from 表名 where 条件;

删除全表数据（截断表）
truncate table 表名 ;

DCL - 数据控制语言

创建用户

create user ‘用户名‘@’ip地址‘ identified by 密码;（%代表任意ip，localhost代表本机）

删除用户

drop user ‘用户名‘@’ip地址‘

授权

  • 授予所有的权限 *.* 所有对象.所有权限

  • 授予用户指定数据库（对象）指定权限

召回权限

revoke all privileges on 数据名.* from ‘用户名‘@’ip地址‘
